void hoverEnterEvent (QGraphicsSceneHoverEvent *) override
 Called when the mouse enters the item. If the item wasn't selected, it will become selected and the fact that it was selected from mouse hover will be stored. This is so that the item cannot be unselected by hoverLeaveEvent if it was selected by group selection.
 
void hoverLeaveEvent (QGraphicsSceneHoverEvent *) override
 Called when the mouse leaves the item. If the item was selected from hover, it will be unselected.
 
void mousePressEvent (QGraphicsSceneMouseEvent *event) override
 Called when the user presses a mouse button. Changes the cursor, and puts the item on top of all other items (for painting purposes).
 
void mouseReleaseEvent (QGraphicsSceneMouseEvent *event) override
 Called when the user releases the mouse button. Changes the cursor and puts the item back the bottom layer (for painting purposes). This method also emits the mouseReleased signal.
 
QVariant itemChange (QGraphicsItem::GraphicsItemChange change, const QVariant &value) override
 Called when the item is changed in any way. Here it is used to check if the item is being moved out of the scene or colliding with another item. If so, it will be stopped from going further.

Detailed Description

The Wall class represents a wall object in the scene.

It inherits from MyItem and QGraphicsRectItem.
